Detailed Description

Implements a CalibrateTask that determines gain and offset errors for all voltages and currents.

Copyright
Copyright 2024 Gregory Aicklen. Licensed under the MIT License

Implements a CalibrateTask that measures sets of raw values from the ADCs of two INA260 voltage/current sensors and computes gain and offset corrections for all measurements. Corrections are written to EEPROM where they can be used by other tasks.

To use the Calibrate task:

  • CalibrateTask::setup() - setup the task
  • CalibrateTask::update() - run the calibration process; call once each time through scheduler. Waits for button presses between steps in the process
  • CalibrateTask::buttonPress() - call to advance to the next step in the process
  • CalibrateTask::finished() - call to determine if calibration process is finished. Scheduling of the Calibrate task should stop when this flag becomes True

Intended to be run by a simple, non-preemptive round robin scheduler, where each task is responsible for relinquishing control; this code is not threadsafe and must not be interrupted.
