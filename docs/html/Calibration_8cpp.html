Detailed Description

Stores and retrieves calibration data to/from EEPROM.

Copyright
Copyright 2024 Gregory Aicklen. Licensed under the MIT License

Implements reading and writing calibration data to/from EEPROM, including checking/computing a CRC for the data to ensure that unwritten or corrupted calibration data are detected and rejected.

If calibration not performed or data corrupt, the following defaults are used for all channels:

  • Voltage Offset: 0
  • Voltage Gain: 1
  • Current Offset: 0
  • Current Gain: 1
