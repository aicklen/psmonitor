void BuzzerTask::update ( void )

Called each time through the scheduling loop to manage the buzzer.

Implements the BuzzerTask. Runs whenever the scheduler transfers control to the BuzzerTask and determines if it is time to run; if not time to run, immediately relinquishes control back to the scheduler.
